## aiAIAI Laravel

### 始め方

```
git clone https://github.com/Soma-dev0808/aiAIAI-laravel.git

php artisan serve
```

### バージョン

* PHP v7.4.11
* Laravel v8.13.0
* node v10.20.1 (NVMのインストールをお勧めします)
